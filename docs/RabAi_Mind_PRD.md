## 一、产品概述

### 1.1 产品定位

RabAi Mind 是一款基于 AI 技术的智能 PPT 生成平台，通过自然语言处理和生成式 AI，帮助用户快速创建专业级演示文稿。平台集成火山引擎 AI 能力，支持从文本描述自动生成完整 PPT。

### 1.2 核心价值主张

- **快速生成**: 用户输入需求，3分钟内完成 PPT 生成
- **质量保证**: AI 驱动的多轮质量检查，确保输出专业水准
- **多端支持**: Web 端 + 小程序双端覆盖，满足不同场景需求
- **格式兼容**: 输出标准 PPTX 格式，兼容 WPS/Office/移动端

### 1.3 目标用户

| 用户类型 | 典型场景 | 痛点 |
|----------|----------|------|
| 职场人士 | 工作汇报、方案展示 | 排版耗时、缺乏设计感 |
| 销售人员 | 产品演示、客户提案 | 模板陈旧、缺乏个性 |
| 教师学生 | 课件制作、学术报告 | 技术门槛高、效率低下 |
| 创业者 | 路演融资、公司介绍 | 缺乏专业设计资源 |

## 三、功能规格

### 3.1 核心功能模块

#### 3.1.1 需求输入模块

| 功能点 | 详细描述 |
|--------|----------|
| 文本输入 | 支持多行文本输入，最少10字符，最多2000字符 |
| 参数配置 | 幻灯片数量(5-30)、场景类型、风格选择 |
| 模板选择 | 提供 3 种以上预置模板 |
| 快捷入口 | 支持模板快速填充示例需求 |

#### 3.1.2 AI 生成模块

| 功能点 | 详细描述 |
|--------|----------|
| 内容生成 | 调用火山引擎 API 生成幻灯片结构化内容 |
| 内容校验 | AI 二次校验内容完整性、专业性 |
| 错误处理 | 生成失败自动重试 3 次 |
| 进度反馈 | 实时返回生成进度百分比 |

#### 3.1.3 SVG 渲染模块

| 功能点 | 详细描述 |
|--------|----------|
| 渲染引擎 | 基于模板的 SVG 页面生成 |
| 尺寸规范 | 16:9 比例，默认 1600x900px |
| 元素布局 | 标题/正文/图片/图表的自动布局 |
| 字体嵌入 | 支持嵌入字体确保一致性 |

#### 3.1.4 PPTX 转换模块

| 功能点 | 详细描述 |
|--------|----------|
| 转换引擎 | SVG → PPTX 格式转换 |
| 格式兼容 | 兼容 Office 2007+ / WPS 2019+ |
| 优化处理 | 文件压缩、兼容性检查 |
| 质量验证 | 打开测试、元素完整性检查 |

#### 3.1.5 接口服务模块

| 接口路径 | 方法 | 功能描述 |
|----------|------|----------|
| /api/v1/generate | POST | 提交生成任务 |
| /api/v1/task/{id} | GET | 查询任务状态 |
| /api/v1/task/{id}/cancel | POST | 取消任务 |
| /api/v1/download/{id} | GET | 下载 PPT 文件 |
| /api/v1/preview/{id} | GET | 预览 PPT |
