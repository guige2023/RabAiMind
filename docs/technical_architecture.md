### 4.2 API密钥安全方案

```python
# 安全配置示例
import os
from functools import lru_cache

class SecurityConfig:
    """安全配置类"""

    @staticmethod
    @lru_cache()
    def get_volcano_api_key() -> str:
        """从环境变量获取火山引擎API密钥"""
        api_key = os.environ.get("VOLCANO_API_KEY")
        if not api_key:
            raise ValueError("VOLCANO_API_KEY 环境变量未设置")
        return api_key

    @staticmethod
    @lru_cache()
    def get_oss_config() -> dict:
        """从环境变量获取OSS配置"""
        return {
            "endpoint": os.environ.get("OSS_ENDPOINT"),
            "access_key": os.environ.get("OSS_ACCESS_KEY"),
            "secret_key": os.environ.get("OSS_SECRET_KEY"),
            "bucket": os.environ.get("OSS_BUCKET")
        }
```

## 六、部署架构

### 6.1 容器化部署

```yaml
# docker-compose.yaml
version: '3.8'

services:
  # FastAPI 主服务
  api:
    build: .
    ports:
      - "8000:8000"
    environment:
      - VOLCANO_API_KEY=${VOLCANO_API_KEY}
      - REDIS_URL=redis://redis:6379
    depends_on:
      - redis
    restart: unless-stopped

  # Redis 缓存/队列
  redis:
    image: redis:7-alpine
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data
    restart: unless-stopped

  # Worker 异步任务
  worker:
    build: .
    command: python -m worker
    environment:
      - VOLCANO_API_KEY=${VOLCANO_API_KEY}
      - REDIS_URL=redis://redis:6379
    depends_on:
      - redis
    restart: unless-stopped

volumes:
  redis_data:
```

## 七、监控与日志

### 7.1 日志规范

```
日志级别:
- DEBUG: 调试信息
- INFO: 正常业务流程
- WARNING: 异常但可处理
- ERROR: 错误需要关注
- CRITICAL: 系统级错误

日志格式:
%(asctime)s - %(name)s - %(levelname)s - %(message)s

示例:
2026-03-17 10:00:00 - rabai_mind.api - INFO - 任务提交成功
2026-03-17 10:00:01 - rabai_mind.worker - INFO - 开始处理任务
```

### 7.2 关键指标

| 指标 | 告警阈值 |
|------|----------|
| API 响应时间 | > 2s 告警 |
| 任务失败率 | > 5% 告警 |
| CPU 使用率 | > 80% 告警 |
