## 需求依赖关系

```
R01 需求输入
  │
  ├─ R01-01 → R02-01 (输入后触发生成)
  ├─ R01-02 → R03-01 (数量影响渲染)
  └─ R01-03 → R03-04 (场景影响模板)

R02 AI生成
  │
  ├─ R02-01 → R03-01 (内容驱动渲染)
  ├─ R02-02 → R04-01 (校验通过才能转换)
  └─ R02-04 → R06-02 (进度反馈前端)

R03 SVG渲染
  │
  └─ R03-01 → R04-01 (SVG作为转换输入)

R04 PPTX转换
  │
  ├─ R04-01 → R05-04 (转换后提供下载)
  └─ R04-02 → R06-01 (兼容才能预览)

R05 接口服务
  │
  └─ 全部为底层依赖

R06 用户体验
  │
  └─ 依赖 R02-R05

R07 MCP协议
  │
  └─ 封装 R05 接口
```

## 优先级定义

| 优先级 | 定义 | 需求数量 |
|--------|------|----------|
